github已经注册了三月有余，除了起初的几个不成气候的项目以外一直没有什么动静。颓废久了便有动动手指的冲动，技术小白的我在关注了周围几个大神以后决定还是要好好利用github这样一个’厉害’的平台，于是乎Github Pages便成为了第一座要攻破的山头啦~~

Github Pages

Github Pages有两种类型：User/Organization Pages和Project Pages,这里我所使用的是User Pages。下面简单总结一下两者的区别：

  1. User Pages是用来展示用户的，而Project Pages是用来展示项目的；
  2. 用于存放User Pages的仓库必须使用username.gitbub.io的命名规则，而User Pages则没有这样的要求；
  3. User Pages将使用仓库的master仓库，而Project将使用gh-pages分支；
  4. User Pages通过http(s)://username.github.io进行访问，而Project Pages通过http(s)://username.github.io/projectname进行访问
